# El Fucho
Difficulty: 900  
Source: [Codeforces — Problem 2155A](https://codeforces.com/contest/2155/problem/A)

This program reads multiple test cases and for each integer `n` prints the value $2(n-1)$.

## Problem (summary)
Given an integer n for each test case, compute and output $2(n-1)$.

## Input
- The first line contains a single integer $t$ — the number of test cases.
- Each of the next $t$ lines contains one integer $n$.

## Output
- For each test case, print a single line with the number $2(n-1)$.

## Example
Input:
2
2
3

Output:
2
4
